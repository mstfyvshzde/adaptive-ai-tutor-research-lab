# Demo Directory

This folder contains the Streamlit demo for the Adaptive AI Tutor Research Lab.

## Main File

The main demo file is:

`app.py`

It can be launched with:

`streamlit run 07_demo/app.py`

## Demo Content

The demo shows:

- project overview
- dataset summary
- supervised model results
- student clustering results
- recommendation examples
- reinforcement learning comparison

## Required Step Before Running

Before launching the demo, run the full pipeline:

`python run_pipeline.py`

This creates the CSV files and plots that the demo displays.

## Purpose

The demo makes the project easier to understand visually.

Instead of only reading code and CSV files, users can explore the main results in a simple dashboard.
