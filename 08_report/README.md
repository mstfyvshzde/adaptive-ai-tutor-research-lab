# Report Directory

This folder contains the written report for the Adaptive AI Tutor Research Lab.

## Purpose

The report explains the project in a structured research-style format.

It describes the problem, method, models, results, limitations, and future work.

## Main Files

- `project_report.md` contains the main project report.
- `appendix/` contains additional supporting materials such as result summaries.

## Summary

This folder turns the technical project into a readable research report.

It explains not only what was built, but also why it matters.
