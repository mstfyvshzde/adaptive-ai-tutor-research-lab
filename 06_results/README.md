# Results Directory

This folder stores the outputs generated by the project pipeline.

## Structure

- `tables/` stores CSV result tables.
- `figures/` stores generated plots.
- `saved_models/` is reserved for future saved model files.

## Generated Outputs

The pipeline can generate outputs such as:

- supervised model metrics
- student cluster summaries
- recommendation examples
- reinforcement learning results
- policy comparison tables
- supervised learning plots
- clustering plots
- reinforcement learning plots

## Reproducibility

Most result files can be regenerated by running:

`python run_pipeline.py`

## Note

This folder represents the evidence produced by the project.

The code is important, but the results show what the system actually produced.
