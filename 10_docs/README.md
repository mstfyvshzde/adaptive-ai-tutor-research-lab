# Documentation Directory

This folder contains supporting documentation for the Adaptive AI Tutor Research Lab.

## Purpose

The documentation folder explains how the project works, how to run it, what limitations it has, and how it could be improved.

## Main Files

- `setup.md` explains how to install and run the project.
- `methodology.md` explains the project method.
- `reproducibility_and_ethics.md` explains responsible use and reproducibility.
- `production_readiness.md` explains what is needed for real-world deployment.
- `future_work.md` explains possible improvements.
- `glossary.md` explains technical terms.
- `model_cards/` contains model and system cards.

## Summary

This folder makes the project easier to understand, reproduce, and evaluate.
