# Adaptive AI Tutor Research Lab

A research-style AI system for personalized learning using knowledge tracing, recommender systems, and reinforcement learning.

## Overview

This project explores how an AI tutor can estimate a student's knowledge state, recommend the next best learning activity, and optimize long-term learning paths.

## Research Question

Can a reinforcement learning-based tutoring policy improve personalized learning paths compared to random, difficulty-based, supervised, and recommender-based baselines?

## Main Components

- Knowledge tracing
- Student behavior clustering
- Recommender system baselines
- Reinforcement learning tutor environment
- Evaluation and ablation study
- Research-style report
- Demo application
