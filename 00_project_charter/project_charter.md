# Project Charter

## Project Title

Adaptive AI Tutor: Personalized Learning with Knowledge Tracing, Recommender Systems, and Reinforcement Learning

## Goal

The goal of this project is to build a research-style AI tutoring system that can analyze student learning behavior, estimate knowledge state, recommend the next best learning activity, and optimize learning paths over time.

## Problem

Most learning platforms give the same content to every student.  
However, students have different weaknesses, learning speeds, mistake patterns, and levels of mastery.

This project focuses on one core problem:

How can AI personalize the learning path for each student?

## Research Question

Can a reinforcement learning-based tutor improve personalized learning paths compared to random, difficulty-based, supervised, and recommender-based baselines?

## Hypothesis

An RL-based tutoring policy can improve long-term learning outcomes because it optimizes learning progress over time instead of only predicting short-term correctness.

## Main Methods

- Supervised learning for knowledge tracing
- Unsupervised learning for student behavior clustering
- Recommender systems for next-task recommendation
- Reinforcement learning for learning path optimization
- Ablation study for system analysis

## Expected Outputs

- Clean GitHub repository
- Dataset analysis
- Knowledge tracing model
- Student clustering analysis
- Recommender baselines
- RL tutor environment
- Evaluation results
- Research-style report
- Demo application
- LinkedIn-ready project materials

## Project Standard

This project is designed as a flagship AI research portfolio project.

The goal is not only to build a working model, but to create a complete research artifact with clean code, experiments, results, visualizations, documentation, and a final report.
