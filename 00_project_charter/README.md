# Project Charter Directory

This folder contains the planning and identity documents for the Adaptive AI Tutor Research Lab.

## Purpose

The project charter explains what the project is, why it exists, what it includes, and how success is measured.

## Main Files

- `project_charter.md` explains the full project idea.
- `research_question.md` defines the main research question.
- `hypothesis.md` explains the expected outcome.
- `scope.md` defines what is included and excluded.
- `success_metrics.md` explains how the project will be evaluated.

## Summary

This folder is the starting point of the project.

It defines the project before the code begins.
