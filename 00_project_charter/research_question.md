# Research Question

## Main Research Question

Can a reinforcement learning-based tutoring policy improve personalized learning paths compared to random, difficulty-based, supervised, and recommender-based baselines?

## Sub-Questions

1. How accurately can a supervised model predict whether a student will answer the next question correctly?

2. Can student learning behavior be grouped into meaningful patterns using unsupervised learning?

3. Which recommendation baseline provides the strongest personalized learning path before reinforcement learning?

4. Can reinforcement learning improve long-term learning gain instead of only optimizing short-term correctness?

5. Which system components contribute most to performance according to ablation analysis?

## Research Focus

This project focuses on adaptive learning systems.

The goal is not only to predict student performance, but to study whether an AI tutor can make better sequential learning decisions over time.
