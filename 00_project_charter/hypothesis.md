# Hypothesis

## Main Hypothesis

A reinforcement learning-based tutoring policy can improve personalized learning paths compared to random, difficulty-based, supervised, and recommender-based baselines.

The reason is that reinforcement learning can optimize long-term learning progress, while many traditional recommendation methods focus only on short-term correctness.

## Educational Hypothesis

Students do not benefit equally from the same sequence of learning activities.

A personalized tutor that adapts to student weaknesses, mastery level, and learning behavior can create more effective learning paths than a fixed curriculum.

## Technical Hypothesis

Combining multiple AI methods will produce a stronger adaptive learning system than using a single model.

The system will combine:

- Supervised learning for performance prediction
- Unsupervised learning for student behavior analysis
- Recommender systems for next-task selection
- Reinforcement learning for long-term learning optimization

## Research Hypothesis

The RL-based tutor will show stronger results when evaluated against multiple baselines, especially in long-term learning gain and mastery improvement.

## Product Hypothesis

A clear and explainable AI tutor can be more useful than a black-box recommendation system.

The system should not only recommend the next activity, but also explain why that activity is useful for the student.

## Evaluation Hypothesis

The project will be considered successful if the RL-based approach performs better than baseline methods in at least one major learning-related metric, such as:

- Learning gain
- Mastery improvement
- Cumulative reward
- Weak-topic improvement
