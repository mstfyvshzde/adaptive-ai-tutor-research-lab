# Project Scope

## Scope of the Project

This project is not designed only for software or computer science learners.

The Adaptive AI Tutor is designed as a general personalized learning system that can support students from different fields, including:

- Mathematics
- Physics
- Biology
- Medicine
- Engineering
- Computer Science
- Language Learning
- Exam Preparation
- Social Sciences
- Any structured learning domain

## Core Idea

The main idea is that every learner has different strengths, weaknesses, learning speed, and mistake patterns.

Because of this, a learning system should not give the same path to every student.

Instead, the system should analyze each learner's progress and recommend the next best learning activity based on their current knowledge state.

## What This Project Includes

This project includes:

- Student performance prediction
- Learning behavior analysis
- Personalized recommendation
- Adaptive learning path optimization
- Explainable AI-based tutoring decisions
- Research-style evaluation

## What This Project Does Not Include Yet

The first version will not try to cover every subject directly.

Instead, it will build the core AI system using student-question interaction data.

Later, the same system can be adapted to different fields such as math, medicine, language learning, or exam preparation.

## Long-Term Vision

The long-term vision is to create an AI tutor that can help learners in any field by understanding what they know, what they struggle with, and what they should study next.

This makes the project broader than a software project.

It is an AI + education + learning science research project.
