# Research Directory

This folder contains research notes and background materials for the Adaptive AI Tutor Research Lab.

## Purpose

The research folder supports the theoretical side of the project.

It includes notes about adaptive learning, knowledge tracing, recommender systems, and reinforcement learning in education.

## Main Files

- `literature_review.md` summarizes the research background.
- `references.bib` stores reference information.
- `paper_notes/` contains short notes about important topics and papers.

## Summary

This folder shows that the project is not only a coding exercise.

It is connected to real AI/ML research ideas in education.
