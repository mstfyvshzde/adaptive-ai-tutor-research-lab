# Experiments Directory

This folder contains experiment planning and tracking documents.

## Purpose

The experiments folder helps track what was tested, why it was tested, and what results were produced.

## Main Files

- `experiment_plan.md` describes planned experiments.
- `experiment_log.md` records experiment progress.
- `experiment_results.md` summarizes important findings.
- `configs/` is reserved for experiment configuration files.

## Summary

This folder makes the project more research-oriented.

It helps separate experiments from source code.
