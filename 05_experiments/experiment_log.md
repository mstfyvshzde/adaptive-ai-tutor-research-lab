# Experiment Log

## Purpose

This file records the experiments, decisions, results, and observations during the Adaptive AI Tutor Research Lab project.

The goal is to keep the project reproducible and research-oriented.

Every major experiment should be logged here with:

- Date
- Experiment name
- Goal
- Method
- Dataset version
- Metrics
- Results
- Notes
- Next action

---

## Log Format

Each experiment will follow this format:

### Experiment ID

### Date

### Goal

### Dataset

### Method

### Metrics

### Result

### Notes

### Next Action
