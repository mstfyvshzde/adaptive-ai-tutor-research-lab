Portfolio assets help explain the project for GitHub, LinkedIn, resumes, demos, and interviews.

- `linkedin_post.md` provides LinkedIn post drafts.
- `github_showcase.md` provides GitHub presentation text.
- `demo_script.md` helps explain the project during a demo or interview.
