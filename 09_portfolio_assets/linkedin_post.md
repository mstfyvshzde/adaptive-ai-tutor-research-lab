## Version 1 — Professional

I built an Adaptive AI Tutor Research Lab project.

The goal of this project is to explore how an AI tutor can personalize learning using student interaction data.

The project combines supervised learning, student clustering, recommendation strategies, reinforcement learning tutor simulation, evaluation, visualization, automated testing, Streamlit demo, research-style documentation, model cards, and ethics discussion.

The full machine learning pipeline is:

data generation → validation → feature engineering → prediction → clustering → recommendation → reinforcement learning → evaluation → demo

This project helped me connect multiple AI/ML concepts into one complete research-style system.

The current version uses synthetic data, and future work includes testing with real educational datasets such as ASSISTments or EdNet.

Tech stack: Python, pandas, NumPy, scikit-learn, matplotlib, Streamlit, pytest, Git, and GitHub.
